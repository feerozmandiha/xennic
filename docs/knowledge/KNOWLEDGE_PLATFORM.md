# Knowledge Platform — پلتفرم دانش مهندسی

**نسخه**: ۱.۰.۰ | **وضعیت**: در حال توسعه

---

## هدف

ایجاد یک پایگاه دانش جامع مهندسی برق که به مهندسان امکان جستجوی هوشمند، دسترسی به استانداردها، و استفاده از دانش فنی در تحلیل‌های مهندسی را می‌دهد.

---

## مؤلفه‌های برنامه‌ریزی‌شده

| مؤلفه | وضعیت | توضیح |
|-------|--------|-------|
| Embedding Engine | ✅ فعال | Sentence Transformers |
| Vector DB (Qdrant) | ✅ فعال | جستجوی برداری |
| RAG Pipeline | ✅ فعال (AI Service) | Retrieval-Augmented Generation |
| استانداردهای فنی | 📋 برنامه‌ریزی | ایندکس کردن استانداردهای IEC, ISIRI |
| Crawler خودکار | 📋 برنامه‌ریزی | خزش وبسایت‌های فنی |
| Dashboard مدیریت | 📋 برنامه‌ریزی | مدیریت محتوای دانش |

---

---

## API (AI Service — فعلی)

| مسیر | توضیح |
|------|-------|
| `/api/v1/ai/knowledge` | جستجوی دانش فنی از طریق LLM |
| `/api/v1/ai/search` | جستجوی برداری در Qdrant |
| `/api/v1/ai/index` | ایندکس کردن سند جدید در Qdrant |

> معماری کامل و جزئیات Ingestion Pipeline در `XENNIC_ARCHITECTURE_SPEC_v1.md` مستند شده است.
