## Getting Started

### Registration
1. مراجعه به `xennic.com`
2. کلیک روی "ثبت‌نام"
3. وارد کردن ایمیل و رمز عبور
4. تأیید ایمیل
5. انتخاب نقش (مهندس / مدیر پروژه / بازبین)

### Workspace Setup
پس از ورود، یک workspace ایجاد کنید:
- New Workspace → نام شرکت/پروژه
- تنظیمات تیم → Invite اعضا
- تنظیمات اشتراک → انتخاب پلن
