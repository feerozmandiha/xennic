9. INCONSISTENCIES FOUND
───────────────────────────────────────────────────────────────────────────────

  1. **ADR Numbering Collision**: Project ADRs 001-010 in docs/decisions/ and
     Knowledge Runtime ADRs 001-010 in docs/knowledge/runtime/ use the same
     numbers for completely different decisions.

  2. **Doc Root Divergence**: docs/ and xennic-docs/ contain 32 files that
     have diverged in content and file count.

  3. **Outdated Status Reports**: DOCUMENTATION_STATUS.md claims 120 files
     when actual is 276. REVIEW_REPORT.md claims 149 files.

  4. **TypeScript Version Mismatch**: Root TypeScript dependency ^6.0.3 vs
     apps/web TypeScript dependency ^5.8.3.

  5. **@nestjs/throttler Everywhere**: All 5 packages include @nestjs/throttler
     as a devDependency despite it being irrelevant to their purpose.

  6. **Package Consumption Inconsistency**: @xennic/database's main points to
     dist/ (compiled) while all other packages point to raw .ts source.

  7. **Naming Collision**: TenantContext is both an interface (in @xennic/types)
     and a class (in @xennic/database).

  8. **Runtime README Broken Reference**: Line 19 references
     governance/acquisition-policy.md but the file is at concepts/.

  9. **Duplicate package.json**: packages/config/src/package.json duplicates
     the outer package.json.

  10. **pnpm-workspace.yaml Corruption**: The allowBuilds section contains
      spaces, quotes, commas, and single letters instead of valid package names.

  11. **Missing ADR Files**: ADR-001 to ADR-005 exist only inline in
      XENNIC_MASTER_ARCHITECTURE.md — no separate files.

  12. **Knowledge FTS**: search_text GIN index exists but no trigger or
      application code populates it automatically.
