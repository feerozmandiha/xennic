# Consultations — سرویس مشاوره

**نسخه**: ۱.۰.۰ | **وضعیت**: فعال | **دامنه**: ماژول `consultations` (NestJS API)

---

## نمای کلی

سرویس مشاوره بستری برای ارتباط مهندسان با متخصصان و مشاوران حوزه برق است. کاربران می‌توانند درخواست مشاوره تخصصی ثبت کرده و با متخصصان مرتبط ارتباط برقرار کنند.

---

## قابلیت‌ها

- ثبت درخواست مشاوره
- تخصیص متخصص
- پیگیری وضعیت مشاوره
- تاریخچه مشاوره‌ها
- رتبه‌بندی متخصصان

---

## API Endpoints

| مسیر | متد | توضیح |
|------|------|-------|
| `/api/v1/consultations` | GET | لیست درخواست‌های مشاوره |
| `/api/v1/consultations` | POST | ثبت درخواست مشاوره جدید |
| `/api/v1/consultations/:id` | GET | جزئیات مشاوره |
| `/api/v1/consultations/:id` | PUT | ویرایش وضعیت مشاوره |
| `/api/v1/consultations/:id/messages` | GET | پیام‌های مشاوره |
